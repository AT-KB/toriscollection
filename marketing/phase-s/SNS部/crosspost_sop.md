## 0. 大原則（4面すべてに共通）

1. **動画本体は1本を使い回す**。差し替えるのは**キャプションとハッシュタグだけ**（＝この規則の目的）。字幕・オンスクリーンのクレジットは焼き込み済みで全面共通。
2. **クレジットは動画内に焼き込む**のを基本とし、キャプションにも録音者/ライセンス/xeno-canto番号を**テキストで併記**する（プラットフォームが動画テロップを圧縮/トリミングしても出典が残るように）。原則4・指示書§5。
3. **静かなトーンを全面で維持**: 絵文字乱用・煽り・CTA連呼・時間圧語（"now only" 等）は禁止（指示書§0-6）。ハッシュタグは"飾り"でなく"発見の導線"。関係ない大量タグ（タグスパム）はしない。
4. **導線は控えめに1回**: キャプション末尾に静かなCTAを一度（`script_templates.md` の共通CTAバンクから）。"pre-register" は使わず "Android closed test"。
5. **比較・ランキングをしない**: 他アプリ/他プレイヤーと比べるタグ・文言は使わない（SNS部禁止事項）。

## 2. キャプション変換テンプレ（同一動画→4面）

同じ1本の動画に対し、**面ごとにキャプションだけ**を下記の型で組む。`{hook}`＝動画冒頭字幕（フックバンク由来）、`{credit}`＝`Recording: [recordist], [license], xeno-canto #[id]`、`{cta}`＝共通CTAバンクから1つ。

### TikTok（短く・掴み優先）
```
{hook}
{cta}
{credit}
#cozygames #birds #relaxing
```
- タグは3–5個。ASOコア語を自然形に（`cozy` `bird` `relaxing`）。生URLは本文に置かず bio リンク枠へ。

### Reels / Instagram（世界観・美しさ優先）
```
{hook}

A quiet, passive bird game — real songs, real ecology. No chores, no rushing.
{cta}
{credit}
```
- ハッシュタグ（3–5個）は**キャプション末尾 or 一次コメント**にまとめる（本文を静かに保つ）。例 `#cozygaming #birdwatching #pixelart #relaxinggames`。

### Shorts / YouTube（検索流入・devlog）
- **タイトル（短く・ASO語前寄せ）**: `{hook} — a cozy bird collector (real recordings)`
- **説明欄**:
```
{hook}
A quiet, passive bird game: grow a tiny ecosystem and real birds arrive to sing.
Songs are real field recordings from xeno-canto, always credited. Food web from GloBI.
{credit}
In Android closed testing now → [LP_URL]
#cozygames #birdgame #relaxing
```

### X（build in public・一人称）
```
{hook}

Real recording (xeno-canto), real food web (GloBI). Building it quietly.
{credit}
Android closed test → [LP_URL]
```
- タグは1–2個まで（例 `#buildinpublic #indiedev`）。長い解説は**スレッドに分割**（1本目に動画＋フック、2本目以降に裏側/出典）。日英併記はXの build in public のみ可（指示書§0-4）。

> **ハッシュタグの芯（ASO連動）**: `aso_keywords.md` のコア5語（cozy bird collector / relaxing bird sounds game / idle bird game / bird collecting game / idle bird collector）を、各社のタグ慣習に合わせて**短い自然形**に崩して使う（`#cozybirdgame` `#birdcollector` `#relaxinggames` `#idlegame` 等）。意図一致を崩す無関係タグは足さない。

### 交渉不能5原則・トーンとの整合
- 原則1（受動的）: 全面で時間圧語・CTA連呼を禁止ルール化。CTAは1回。
- 原則3（声と癒しは無料）: CTAに"売らない"選択肢を許容。課金で癒しを条件化する文言なし。
- 原則4（生態に誠実）: 全面でクレジット必須（動画＋キャプション二重）。出典なし録音使用をNGに明記。
- 原則5（かわいさ最優先）: 面ごとに世界観（庭・鳥・声・ドット絵）を主役に置くトーン指定。
- SNS部禁止事項: 他アプリ/他プレイヤー比較タグ・ランキング表現なし。タグスパム禁止。
