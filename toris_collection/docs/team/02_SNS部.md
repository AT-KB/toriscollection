# SNS部向け — 方針とルール

対象: 拡散・コミュニティ・ストア露出(ASO)・コラボ発信を担う人。

## SNSのミッション
**「鳥のかわいさと癒し」を外に広め、"本物の鳥への興味"の入口になる。**
売り込みでなく、見た人が「かわいい/落ち着く/へぇ(学び)」と感じて自然に共有したくなる形。

## 発信の芯(何を伝えるか)
1. **かわいい**: 来る瞬間、ドット絵、鳴き声。
2. **癒し**: 環境音+ときどきの鳥。作業/就寝BGM文脈と相性◎。
3. **学び(軽く)**: 「なぜこの鳥が来たか」を一言で。押しつけない豆知識。
4. **習慣**: 「今日も◯◯が来た」「会った日数◯日」= 日々のささやかな物語。

## SNSのルール(交渉不能)
- **誇大表現をしない**。「鳴き声が聴ける」は事実ベース(録音は xeno-canto、要クレジット)。
- **鳥の声は無料**という約束を発信でも守る(課金誘導を主役にしない)。
- **録音者・出典のクレジット**を尊重(素材の権利表記。商用可の範囲で運用)。
- **保全は"押しつけない"**。募金強制はしない。「興味を持つきっかけ」に留める。
- **他者・他アプリを貶さない**。癒し系の世界観を発信のトーンでも保つ。

## 使える発信フォーマット(今の資産ベース)
| フォーマット | 中身 | 状態 |
|---|---|---|
| 共有カード | 「今日、庭に◯◯が来ました」+ ドット絵 + 一言 | 企画中(自動生成が理想) |
| 鳴き声つき短尺 | 15秒の癒しクリップ | 将来 |
| 「今日の鳥」 | 実在情報の豆知識 + かわいい絵 | 既存の"今日の庭"を発展 |
| みんなの庭 | 匿名の集合地図「誰かの庭に今日来た鳥」 | 実装済(発信素材化) |
| 会った日数 | 「◯◯に会って30日・常連になった」物語 | 実装済 |

## ASO / ストア(Google Play 想定)
- キーワード方針: 「鳥 癒し 放置 かわいい 鳴き声 生態系 バードウォッチング」。
- スクショ/説明は **3本の柱(構築SIM/声/癒し)** を1枚ずつ。「庭が無くても遊べる」を明記。

## 中期のコラボ(SNSが橋渡し)
- 野鳥の会・博物館・保全団体・光学機器ブランドとのタイアップ発信。
- 「生態が学べる」信頼が武器。教育文脈での紹介を狙う。

## 判断に迷ったら
「この投稿は**かわいい/癒し/軽い学びのどれかを正直に伝えているか? 誇張・課金誘導・権利無視に
なっていないか?**」を確認。No があれば出さない。
